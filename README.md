## How It Works

- Each session starts with a **daily recap quiz** on the previous day's concepts
- Concepts are explained in plain English before any code is written
- Each day produces a standalone script named `dayNN-topic.sh`
- `LEARNING_LOG.md` tracks key insights and quiz questions session to session

## Files

- `LEARNING_LOG.md` — running log of concepts, insights, and quiz results
- `dayNN-topic.sh` — one script per day
